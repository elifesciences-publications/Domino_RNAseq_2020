## Summary

This repository contains analysis code for the manuscript "Drosophila SWR1 and NuA4 complexes originate from DOMINO splice isoforms" by Scacchetti et al. 2020, eLife https://doi.org/10.7554/eLife.56325

## Design
RNAseq experiment of control (GST/GFP), H2A.v, Domino-A, Domino-B and Tip60 RNAi in 3 biological replicates.
